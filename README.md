# 项目简介 java134

本项目是一款基于前后端分离架构的游戏管理系统，采用 Spring Boot、Vue、ElementUI、MySQL 和 Redis 技术栈进行开发。

## 技术栈

- 后端：Spring Boot
- 前端：Vue + ElementUI
- 数据库：MySQL
- 缓存：Redis

## 功能模块

### 普通用户

- **注册**：输入用户基本信息进行账号注册
- **登录**：进入游戏商城
- **系统主页**：推荐本周最佳交易游戏和最新游戏，支持通过游戏名称搜索
- **系统公告**：查看系统公告
- **留言咨询**：为系统留言咨询
- **购物车**：将游戏添加至购物车
- **我的收藏**：将游戏添加至个人收藏
- **下单购买**：选择游戏，进入支付页面进行下单购买
- **我的订单**：查看历史购买订单
- **地址管理**：编辑和管理收货地址
- **修改个人信息**：更新用户个人信息

### 管理员

- **登录系统后台**：使用管理员账号登录后台管理系统
- **首页**：汇总统计最近的游戏销售额情况
- **游戏管理**：
  - 游戏分类管理
  - 新增游戏
  - 上下架游戏
  - 修改游戏信息
  - 删除游戏
- **系统公告管理**：设置系统公告
- **留言咨询管理**：查看及回复留言咨询
- **订单管理**：管理系统所有订单
- **配送员管理**：进行配送员管理
- **系统轮播图管理**：管理系统首页轮播图
- **用户管理**：管理注册用户
- **个人中心**：编辑修改管理员个人信息
- **系统设置**：
  - 管理员管理
  - 角色管理
  - 菜单管理
  - 日志管理

## 运行环境

- 后端：Java 1.8+
- 前端：Node.js
- 数据库：MySQL 5.7+
- 缓存：Redis 3.2+
